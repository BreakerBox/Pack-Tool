# Pack-Tool
Es una multi herramienta para termux
que te sirve de mucho

# Instalacion
# Required gems

 Ascii85
 Base64
 Base32
 Base16
 Lolcat
 Rchardet
 Paint
 FileUtils
 Open-Uri
 Git

# Installation in termux

$ apt update

$ apt upgrade

$ apt install git

$ apt install ruby

$ git clone https://github.com/BreakerBox/Pack-Tool.git/

$ cd Pack-Tool

$ ruby install.rb

# installation in Kali Linux

$ git clone https://github.com/BreakerBox/Pack-Tool.git/

$ cd Pack-Tool

$ ruby install.rb

# Update

cd Pack-Tool

ruby pack-tool.rb --update
